## Getting Started

First install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

After, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Please, don't forget to add your own .env files.
NEXT_PUBLIC_API_HOST env variable is required.
